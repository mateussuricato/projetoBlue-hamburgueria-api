# API usando nestJS projeto Blue

## Como fazer o clone

```bash
$ git clone git@github.com:mateussuricato/projetoBlue-modulo4.git
```

## Como instalar as dependências

```bash
$ npm install
```

## Como rodar o projeto

### Produção

```bash
$ npm run start
```

### Desenvolvimento

```bash
$ npm run start:dev
```
